# Vocabulário de Tags

> Vocabulário controlado do vault Noosphere. Use APENAS as tags listadas abaixo. Tags novas só entram após auditoria.

## Status (obrigatória em toda nota de paper)

| Tag | Descrição |
|-----|-----------|
| `#inbox` | Captura inicial. Paper recém-chegado, ainda não lido com atenção |
| `#lendo` | Leitura em andamento. Anotações parciais |
| `#concluido` | Leitura completa. Crítica preenchida, conexões feitas |
| `#referencia` | Paper de consulta rápida (metodologia, revisão). Não lido na íntegra |

## Domínio

| Tag | Descrição |
|-----|-----------|
| `#carbono` | Estoques, fluxos e dinâmica de carbono em florestas |
| `#redd` | REDD+, pagamentos por serviços ambientais, MRV |

| `#floresta` | Ecologia florestal, fitossociologia, sucessão |

| `#desmatamento` | Detecção, drivers, trajetórias de desmatamento |
| `#mercado-de-carbono` | Mercados regulados e voluntários, créditos de carbono |
